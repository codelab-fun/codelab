Introduction

Every application starts out with what seems like a simple task: get data, transform them, and show them to users. Getting data could be as simple as creating a local variable or as complex as streaming data over a WebSocket.

An Angular pipe is a function that transforms input values to output values for display in a view. Let's see how it works.

How do pipes work?

A pipe takes in data as input and transforms it to a desired output.

Inside the interpolation expression, we flow the component's dob value through the pipe operator ( | ) to the Date pipe function on the right. Finally we get a more human friendly output. All pipes work this way.

Built-in Pipes

Angular comes with a stock of pipes which are available for use in any template or binding expression. Some of these pipes are:

  • DatePipe
  • UpperCasePipe
  • LowerCasePipe
  • CurrencyPipe
  • PercentPie
