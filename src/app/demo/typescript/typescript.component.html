TypeScript

  • TypeScript is a language that transpiles to JavaScript. It extends ES5 standard (current version of JavaScript supported by modern browsers)
  • It adds features from the next version - ES6 (mostly supported by modern browsers)
  • On top of it, it adds an optional Type System and decorators (we'll learn about them later)

TypeScript / Types / Primitives (strings, numbers, etc...)

Types are defined as variableName: type

const is a new way to declare variables (introduced in ES6). Similar to var, but has block scope, and shows an error if the value is reassigned.

Arrays are defined as Array<Type> or Type[]

TypeScript / Classes

TypeScript has classes, and Angular uses them heavily.

They are similar to classes in other languages, and are used to group methods and properties together

TypeScript / Classes / Constructor

There's a special method on the class called constructor. It's run when the class is instantiated and allows the class to take parameters

TypeScript / Classes / Access Modifiers

Constructor parameters marked as public (or private, or protected), become class properties accessible as this.<parameter name> within the class

private or protected properties are not visible outside of the class.
