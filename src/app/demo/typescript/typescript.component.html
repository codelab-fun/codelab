TypeScript

  • TypeScript a language transpiled to JavaScript. It extends ES5 standard (current version of JavaScript supported by modern browsers.)
  • Adds features from the next version - (ES6)
  • On top of it adds an optional Type System and decorators (we'll learn about them later)

TypeScript / Classes

TypeScript has classes, and Angular uses them heavily

They are similar to classes in other languages. Used to group methods and properties belonging to one object together

TypeScript / Classes / Constructor

There's a special method on the class called constructor. It's run when the class is instantiated and allows the class to take parameters

TypeScript / Classes / Access Modifiers

Constructor parameter marked as public (or private, or protected), becomes a class property accessible as this.name within the class

private or protected properties are not visible outside of the class.
