What is Angular?

Angular is a development platform for building mobile and desktop applications. Angular lets you extend HTML's syntax to express your application's components clearly and succinctly. Angular's binding and Dependency Injection eliminate much of the code you would otherwise have to write.

Given an HTML file

Let's create an Angular app which replaces <hello-world> element with the application contents.

This can be done with 3 simple steps.

Creating Angular app

Takes 3 steps:

  1. Create an Angular component
  2. Create an Angular module
  3. Bootstrap the module

Step 1: Create component

Angular component is just a class. Properties and behavior can be added inside.

Component Anatomy / Decorators

The class is adorned with a @Component decorator. Decorators allow to specify Angular-specific information without altering the actual class.

Decorators

Decorators are a new feature of TypeScript.

Decorators are inspired by a similar feature in Python language.

Component Anatomy / Selector

Selector defines the location of the component. When Angular renders this component, it'll find a <hello-world> HTML element in the document and will render the component inside of it.

Component Anatomy / Inline Template

Template defines the HTML code that the component generates.

If the amount of HTML grows out of hand, it's possible to use templateUrl instead and provide a path to a HTML file.

NgModules help organize an application into cohesive blocks of functionality.

Module Anatomy / Class

Like a component, Angular module is just a class.

Module Anatomy / Decorator

Like a component, Angular module is adorned with a decorator providing metadata.

Module Anatomy / Browser Module

Because we're building a browser web app, we need to pass BrowserModule to the imports array.

Angular is not just for web apps anymore, it allows to create native phone apps and even VR scenes.

Module Anatomy / Declarations

Declarations array, specifies components belonging to the AppModule.

Module Anatomy / Bootstrap

Component passed into the bootstrap array will be created and displayed in your index.html file.
